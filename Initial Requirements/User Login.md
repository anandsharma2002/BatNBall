# User Sign up and Log in

## Process

### Sign up

1. Only Super Admin can create new account (Signup) for users.

2. Phone Number + Password
    - Do not need OTP to Sign up.
    - Do not need email to Sign up.
    - Do not need verification to Sign up.

3. After Sign up
    - User can login using Phone Number + Password
    - User can update profile later
    - Super admin can delete account anytime.

### Log in

1. Phone Number + Password
2. User can update profile after Login
3. User can use all the feature of application after login

### Change Password
1. Current Password + New Password + Confirm New Password


### Forgot Password
1. Phone Number + OTP + New Password
