# Create New Match

## Process
1. User A clicks on New Match button
2. New Tab will be opened for
    - First Team Name
    - Second Team Name
    - Cancel button
    - Next button

3. After clicking on next button,
    - Share Button: User can share a link on this match on any other platform like whatsapp, instagram etc.
    - First Team Name
    - First Team Player list (live joining or adding)
    - Second Team Name 
    - Second Team Name list (Live Joining or adding)
    - Add Player button for both teams: User will clicks on First team add button and search player and select to add in team. And just like for second team.
    - Remove User Button for both teams
    - Choose Captain
    - Number of Overs
    - Choose Umpire: User can select upto 4 umpires. Only umpires can edit the live match score and stats. Other members can only view the live match details. [Importatnt: User him/herselft will be a default umpire, so he can edit the match score, add score etc. Other then user upto 3 umpires (Total Maximum 4)]
    - Next Button
    - User those have the link can join the match and choose team first or second by selecting team and clicking on join
    - After click on join button user will be added in the team list. And user can see the list of players those are added in the match. (live tracking)

4. By clicking on next button, the Toss screen will be visible
    - Toss win:  First Team Name, Second Team name. User have to choose on option
    - Toss Decision: Bat or Field.
    - Next Button

5. By Clicking next button match will start.
    - A Main Score screen will be visible.
    - A share button, Anyone can see the live match tracking and scroe updates without logging using this link. 

6. Live match view/edits
    - Umpire / User can select the batsman (stricker and non striker) and bowler.
    By confirming the selected batsman or bowler the batsman and bowler will be selected.
    - Other members will only view the match details. (live tracking)
    - There should be automatically tracking of batsman, stick rotation, number of overs and ball, extra balls, wickets, runs. It means the User experience of Umpire should be very smooth and user friendly with minimun button click work. Almost everything must be automatically calculated. Like if a player hits 1 run, automatically rotate stick, add 1 run, change strike batsman to non striker and non striker to other batsman. that run automatically added to the batsman, bowler and team. 
    - If wicket is happen then remaining batsmen list should be opened for select batsman (player) with confirmation.
    - If any edit happend by mistken the their should be undo till 5 back steps. It will helps to remove the mistacks. 
    - If user mistakenly choosed wrong bowler or batsman then it also can be editable in live match. And it should also update the stats and record of batsman and bowler.
    - Each and eveything much be working in live update. every player of user of the application can able to see match live update instently.
