# UI Colors 

## Light Mode
1. Dominant (60%): #FDFCF5 (Cream/Off-white)
2. Secondary (30%): #1D4F2A (Classic Cricket Green)
3. Accent (10%): #C6A567 (Soft Gold)
4. Text (High Contrast): #1A1A1A (Near Black)

 
## Dark Mode
1. Dominant (60%): #1A1A1D (Dark Charcoal)
2. Secondary (30%): #4E4E50 (Stone Grey)
3. Accent (10%): #C39797 (Dusty Rose Gold)
4. Text (High Contrast): #EAEAEA (Light Grey)

## Devices:
1. Mobile: Most important device width. All the user mostly (99%) open this application website on mobile so the primary device width and UI and css should be focused on mobile device.
